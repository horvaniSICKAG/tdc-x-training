2.2 Read DIOs
- Get back to Visual Studio /myapp/main.go from task 1
- Create folder pkg
- Get dio.proto from git or documentation on TDC-X and put it in pkg folder
- run commands to get protobuf compiler
go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest
export PATH="$PATH:$(go env GOPATH)/bin"
apt install -y protobuf-compiler
- Generate Proto Code
protoc --go_out=pkg --go-grpc_out=pkg pkg/dio.proto
- Go back to main.go (either use Copilot - prompt below, or copy main_1.go and run command to build
Generate code that connects to /var/run/hal/hal.sock and create a function that sets DIO_D to high state. In main function call that once at start. 
Then generate a function that reads all other inputs and outputs and outputs them as a joined json format in one line, with states 0 and 1 respectivly.
Make that read function execute every 1 second.

Connect cable, find out which is DIO_D and connect to other inputs and watch other inputs change from 0 to 1
