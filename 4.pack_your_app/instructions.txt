4. Pack Your Application to be used in TDC-X application store

When we want to build application for application store, we should check documentation (swagger) of control center POST application.
NOTE:logo.png was generated by AI, You can generate Your own 100x100px .png picture.

Commands to use to build docker:
docker build -t leaf:latest .
docker save -o leaf.tar leaf:latest

Commands to pack all into an application:
tar -czvf leafapp.tar.gz compose/ images/ VERSION config.json logo.png
